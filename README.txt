A test of Bot::Backbone to build a jabber bot.

Goals:
* jabber:
    * connect to jabber server as user
    * add a room connection
    * listen for commands
        * gerrit Change-ID <-> review #
        * gerrit review status
* http:
    * receive github webhook format
    * listen for REST commands to send information from Gerrit
* ssh:
    * query Gerrit for repository information
* git:
    * alternative, query a git repo for desc


Sulla:
    Name of the secretary robot in Rossum's Universal Robots.  Helena can't believe Sulla is a robot.
    Roman dictator with the "cunning of a fox and courage of a lion."  He was dictator of Rome, used his powers to reform the constitution to restore the balance of power between the Senate and Tribune and then resigned the dictatorship and returned constitutional government.
    Celtic goddess Sulis is sometimes called Sulla.
